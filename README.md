## Getting Started

1. Make sure you have [NodeJS](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed.
2. Make sure  mongodb is running.
3. Install your dependencies

    ```
    npm install
    ```
4. Start your app

    ```
    npm start
    ```
   
   For continuous coding without restarting the app use
   ```
   npm run start:dev
   ```
